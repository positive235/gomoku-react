# GOMOKU in React JS

This is a personal project: Traditional Gomoku game(five-in-a-row) played by 2 players. 

## Where to try

https://positive235.github.io/gomoku-react/

## How to run

```
cd gomoku-react
cd gomoku
npm i
npm start
```
